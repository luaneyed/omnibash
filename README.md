# Omnibash #

### Prerequisites ###
- bash
- fingers or toes

### Easy Installation ###
```
$ git clone https://github.com/sangguk2/omnibash.git --depth 1
$ . omnibash/init.sh
```
You must **not** edit the auto generated line in ~/.bash_profile
